## **SECTION 3: WORK TAB (Task Management)**

### **3.1 View Tasks**

**Navigate:** Tap **Work** tab at bottom

**Filter Options:**
- **All** (15) — All tasks
- **Open** (3) — Not yet started
- **In-progress** (5) — Currently active
- **Completed** (4) — Done
- **On-hold** (3) — Paused/waiting

**Each task shows:**
```
┌─────────────────────────────┐
│ Task Title: Safety Inspection
│ 👤 Assigned: Rajesh (SE)
│ Created by: PM Vikas
│ Due: 28 Mar 2026
│ Priority: 🔴 High  Status: 🟡 Open
└─────────────────────────────┘
```

### **3.2 Open a Task (View Details)**

**Step 1:** Tap any task card

**Step 2:** Full task details open:
- Title, description, priority
- Assigned to (team member)
- Due date & creation date
- Status (Open / In-progress / Completed / On-hold)
- Comments & updates
- Attachments (if any)

**Step 3:** Update status (if you're assigned):
- If **Open** → Tap "Start" → Changes to **In-progress**
- If **In-progress** → Tap "Complete" → Changes to **Completed**
- If stuck → Tap "On-hold" → Add reason

### **3.3 Create New Task (PM/SM Only)**

**Step 1:** Tap **[+ Create Task]** button (only visible if you're PM or SM)

**Step 2:** Fill in:
- **Title** (e.g., "Install safety scaffold")
- **Description** (details, steps)
- **Assign To** (select team member)
- **Priority** (Low / Medium / High / Critical)
- **Due Date** (pick from calendar)

**Step 3:** Tap **[Create]** → Task created, assigned person gets notification

**Features:**
- 📅 Calendar view (current month)
- Each day card is expandable

**On each day card:**
- Date & day name (Monday, Tuesday, etc.)
- Planned availability (On-site, WFH, Leave, Field, Off, Half-day)
- Worker count for that day

**Step 1:** Tap day card to expand

**Step 3:** To change your status:
- Tap **[Edit Schedule]**
- Select your status: On-site / WFH / Leave / Field / Off / Half-day
- Tap **[Save]**

### **6.2 Team Subtab — Weekly Availability Grid**

**View:** 3-day, 7-day, or 14-day team availability

**Color coding:**
- 🟢 Green = On-site
- 🔵 Blue = WFH
- 🔴 Red = Leave
- 🟠 Orange = Field
- ⚪ Gray = Off/Half-day

**Shows:** Your personal attendance history (month-by-month)

- Date, IN time, OUT time
- Total hours worked
- Status (Present/Absent/Leave)

### **6.4 Requests Subtab — Mispunch/Extra Hours**

**For handling corrections:**

**Mispunch Request:**
- "I came in 15 mins late, system shows I arrived at 9:30 but it was 9:15"
- Click **[Request Correction]**
- Select date, correct time
- Submit reason
- Supervisor approves/rejects

**Extra Hours Request:**
- "I worked until 8 PM today (2 extra hours)"
- Click **[Request Extra Hours]**
- Date, hours worked, reason
- Supervisor approves (for future payroll)

## **SECTION 10: TROUBLESHOOTING**

### **Problem: Can't log in**

**Possible causes:**
- Wrong username/password → Reset password (link on login screen)
- Account not activated → Contact HR
- Browser cache issue → Clear cookies & try again

**Solution:**
1. Tap **[Forgot Password]** on login
2. Enter your username
3. Check email for reset link
4. Create new password
5. Login with new password

---

### **Problem: Messages not showing**

**Possible causes:**
- Offline (no internet)
- Messages still loading
- Browser issue

**Solution:**
1. Check internet connection (WiFi/mobile data)
2. Refresh page (F5 or Ctrl+R)
3. Wait 10 seconds for messages to sync
4. Try different browser

---

### **Problem: Can't see other employees**

**Possible causes:**
- Wrong tab (check you're on "Internal" or "External")
- Search filter active (clear search bar)
- No employees exist yet in system

**Solution:**
1. Go to **Crew** → **Internal** (or External)
2. Clear search box
3. Scroll down to see all employees
4. Try different filter (role, status)

---

### **Problem: Report blank/won't generate**

**Possible causes:**
- No data in system
- Browser blocking pop-ups
- PDF generator issue

**Solution:**
1. Ensure data exists (check dates are correct)
2. Allow pop-ups in browser settings
3. Try Chrome browser (recommended)
4. Refresh page & try again

---

### **Problem: Sticky + button disappeared**

**Possible causes:**
- You don't have permission to add employees/contacts
- Bug (rare)

**Solution:**
1. Check your role (PM/SM/HR can add)
2. Refresh page
3. Clear browser cache
4. Contact support
