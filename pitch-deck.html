07 — User Manual

Getting Started — First Time Setup

1

Install the APK

Download and install the SiteSync APK on your Android phone. No Play Store needed — direct install. Allow "Unknown Sources" in Settings → Security if prompted.

2

Login with your credentials

Enter your username and password provided by your Project Manager or HR. First-time users should change their password from the Profile section.

3

Mark your attendance

From the Home (Dashboard) tab, tap the large Mark IN button when you arrive at site. Tap Mark OUT when leaving. Your attendance is saved instantly.

4

Set your availability

Go to More → Schedule → Plan tab. Tap on any day card to set your status: On Site / WFH / Leave / Field / Off / Half Day. Your team can see this in real time.

Attendance — How It Works

Employee Attendance

  • Tap Mark IN on Home dashboard when arriving
  • Tap Mark OUT when leaving — time is recorded
  • Multiple IN/OUT pairs allowed (if you leave and return)
  • Cannot mark IN twice without marking OUT first
  • View your history: More → Schedule → History

Mispunch / Correction

  • Forgot to mark IN or OUT? Submit a Mispunch Request
  • Go to More → Schedule → Requests → + New Request
  • Select date, provide correct times + reason
  • PM / SM / HR will approve or reject
  • Approved requests auto-apply to your record

Labour Management System (LMS) — For Supervisors

1

Enroll a New Labour

Open LMS app → tap Enroll Worker → enter name + mobile number. A unique 7-digit Worker ID is auto-generated. Multiple workers can share one mobile number (companions).

2

Mark Labour Attendance

Search worker by ID or name → tap Mark IN. At end of day, tap Mark OUT. System prevents double IN or OUT without IN.

3

Generate Reports

View daily / weekly attendance reports. Export to verify against contractor's muster roll. Prevents fake attendance fraud.

Reports — PM / SM / HR Only

Available Reports

  • Attendance Report — daily/monthly present/absent per employee
  • Availability Report — who was On Site vs WFH vs Leave
  • Labour Summary — total labour hours per day
  • Seniority Report — experience ranking of all staff
  • Scanner Log — who scanned labour IN/OUT

How to Print / Save PDF

  1. Go to More → Reports
  2. Select report type and date range
  3. Tap Generate
  4. Tap 🖨️ Print / Save PDF
  5. Use Chrome's Share → Print or Save as PDF
