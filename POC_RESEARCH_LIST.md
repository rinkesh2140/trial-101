## **HOW TO USE THIS LIST**

### **Phase 1: Research (20 mins per company)**
1. Search company on **Google**, **LinkedIn**, **IndianSME**
2. Find founder/MD contact on LinkedIn
3. Find site address (recent projects)
4. Find domain pain point
5. Note contact & decision-maker

### **Phase 2: Cold Outreach (Email/WhatsApp)**
- Share **5-line pitch** (problem → solution → ROI)
- Offer **30-day free trial** 
- Include **demo video link** from your app

### **Phase 3: Demo Call**
- 20-min call (video or in-person on site)
- Show real dashboard
- Ask about current pain points
- Propose pilot on **1 site**

### **Phase 4: Pilot (30 days)**
- Implement on 1 project
- Track ROI metrics
- Get testimonial
- Convert to 2nd site

## **SALES SEQUENCE (30-60 days)**

### **Day 1: Cold Outreach**
- Send email + WhatsApp message
- Include demo link
- Wait for response (24-48 hours)

### **Day 3: Follow-up**
- If no response, send one follow-up message
- Add value (e.g., "Here's how other companies saved money…")

### **Day 5: Call**
- Call directly (reference previous email/message)
- Keep it brief (2 mins)
- Ask: "Can we schedule 20-min demo?"

### **Day 8-10: Demo Call**
- Show live dashboard
- Show how attendance is recorded
- Show a sample report
- Ask about their pain points
- Propose pilot on 1 site

### **Day 15: Pilot Start**
- Set up on their test site
- Train supervisors (30-60 mins)
- Support daily for first week

### **Day 45: Review Results**
- Show metrics (attendance accuracy, time saved)
- Calculate ROI
- Ask: "Ready to scale to 2nd site?"

### **Day 60: Contract**
- If successful, sign agreement
- Move to full implementation
