## **SLIDE 1: PROBLEM STATEMENT**

### Current Pain Points in Construction Labor Management:

❌ **Manual Attendance** → Errors, fraud, time-consuming  
❌ **No Real-Time Visibility** → Don't know who's on-site, how many workers  
❌ **Labor Data Scattered** → Excel sheets, WhatsApp, fragmented info  
❌ **Delayed Reporting** → Days to generate attendance/payroll reports  
❌ **No Compliance Tracking** → Difficult to track compliance, leaves, approvals  
❌ **Duplicate Records** → Same workers registered multiple times  

**Impact:**
- ₹15,000-50,000 lost per month per project (fraud, inefficiency)
- 3-4 hours/week lost on manual data entry
- Delayed wage payments → worker dissatisfaction
- No audit trail for compliance/insurance

---

## **SLIDE 2: THE SOLUTION — SiteHub**

### **What is SiteHub?**

A mobile-first, cloud-based platform for real-time labor attendance & site management.

**Three Apps in One:**
1. **Labor App** (for workers) → Simple IN/OUT attendance
2. **Site Manager** (for supervisors) → Verify attendance, manage workers
3. **Dashboard** (for project managers/admin) → Reports, analytics, decisions

**Core Features:**
✅ Real-time attendance (GPS-verified on mobile)  
✅ Worker profiles & unique IDs  
✅ Companion worker support (multiple workers per phone)  
✅ Instant reports (daily, weekly, monthly)  
✅ Leave & approval tracking  
✅ Mispunch request system  
✅ Team availability planner  
✅ Multi-project support  

---

## **SLIDE 3: HOW IT WORKS — 3 Simple Steps**

### **For Workers:**
1. **Open App** → Tap "IN" when arriving
2. **System Records** → Time, GPS, photo/face recognition (optional)
3. **Confirm "IN"** → Get notification

### **For Site Managers:**
1. **Review Attendance** → See live dashboard who's on-site
2. **Verify Entries** → Accept or reject mispunches
3. **Generate Reports** → One click, send to PM

### **For Project Managers:**
1. **Login Dashboard** → See all sites/teams
2. **Track Metrics** → Attendance %, leave trends, productivity
3. **Make Decisions** → Allocate resources, approve leaves, pay workers
